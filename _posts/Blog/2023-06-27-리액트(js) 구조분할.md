<h2>구조분해할당</h2>

구조분해할당이란 변수나 객체의 프로퍼티를 변수안에 저장해서 사용하는 방식 을 말한다

```
const numbers = [1,2,3];
[num1,num2] = numbers;
console.log(numbers) //  1,2

[num1, , num2] = numbers;
console.log(numbers) // 1,3

{name} = {name:'Lee',age:28}
console.log(name); // 'lee'
console.log(age); // undefined
```
